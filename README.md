### Iniciando mi proyecto👩🏻‍💻

- 2 Crear en src un archivo .env y pegar "REACT_APP_FIREBASE_API_KEY = AIzaSyAxCKty6DWXXRGUJwGWSvCqreHNsCThWQA" SIN COMILLAS;
